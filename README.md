# 📌 Estrutura de Vetores em JavaScript - AT1.4

## 📖 Descrição
Este repositório contém atividades práticas desenvolvidas em JavaScript com foco na utilização de vetores para armazenamento e manipulação de dados. Os exercícios envolvem cadastro de informações, análise de dados e uso de estruturas de repetição para percorrer vetores sem utilizar funções próprias de arrays.

## 🎯 Objetivo

- Praticar a declaração e manipulação de vetores
- Armazenar diferentes tipos de dados em vetores
- Utilizar estruturas de repetição para percorrer vetores
- Realizar comparações e análises de dados armazenados
- Identificar valores máximos e mínimos em vetores
- Trabalhar com cadastro de pessoas e funcionários
- Utilizar versionamento com Git e GitHub

## 🛠️ Tecnologias Utilizadas

- JavaScript
- Visual Studio Code
- Node.js
- Git e GitHub
- Navegador Web

## 📂 Estrutura do Projeto

O projeto contém 3 algoritmos em JavaScript:

### 1️⃣ Cadastro de Idades
- Leitura da idade de 4 pessoas
- Armazenamento das idades em um vetor
- Identificação da maior idade digitada
- Exibição da posição onde a maior idade foi informada

### 2️⃣ Cadastro de Funcionários
- Leitura do nome, sexo e salário de 5 funcionários
- Armazenamento dos dados em vetores separados
- Exibição apenas das funcionárias mulheres com salário superior a R$ 5.000,00

### 3️⃣ Cadastro de Alturas
- Cadastro de nomes e alturas de várias pessoas
- Encerramento do programa apenas quando o usuário desejar parar
- Identificação da maior e da menor altura cadastradas
- Exibição dos respectivos nomes das pessoas associadas a essas alturas

## 📌 Resultados Esperados

- Vetores corretamente declarados e preenchidos
- Armazenamento eficiente dos dados informados pelo usuário
- Uso adequado de laços de repetição para percorrer vetores
- Comparação de valores para identificação de máximos e mínimos
- Exibição correta das informações solicitadas
- Código funcional, organizado e bem estruturado

## 📌 Critérios Trabalhados

- Manipulação de vetores

- Entrada e saída de dados

- Lógica de programação

Atividade desenvolvida para reforçar conceitos de lógica de programação, armazenamento de dados em vetores, análise de informações e utilização de estruturas de repetição em JavaScript sem o uso de funções específicas para arrays.
